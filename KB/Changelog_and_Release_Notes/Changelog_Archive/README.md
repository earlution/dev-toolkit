# Changelog Archive

This directory contains **detailed changelog entries** for each version of the `vibe-dev-kit` repository.

## Contents

Each file is named using the pattern:

- `CHANGELOG_vRC.EPIC.STORY.TASK+BUILD.md`

Examples:

- `CHANGELOG_v0.1.3.1+1.md`
- `CHANGELOG_v0.4.3.7+1.md`
