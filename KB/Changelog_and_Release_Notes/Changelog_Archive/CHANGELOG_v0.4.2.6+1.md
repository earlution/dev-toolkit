## Summary

This release implements GitHub Issue integration for submitting Feature Requests (FR), Bug Reports (BR), and User Experience Research (UXR) reports. External contributors can now submit via GitHub Issues, which are automatically converted to Kanban documents and processed through the intake workflow.
