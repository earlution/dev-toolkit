## Summary

This release fixes BR-003 by adding support for `update/*`, `maintenance/*`, and `upgrade/*` branch patterns in the Release Workflow branch validation script. Maintenance/update branches now skip epic/version validation (similar to `main` branch), eliminating false warnings when running RW on framework update branches like `update/ai-dev-kit`.
