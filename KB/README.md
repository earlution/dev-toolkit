# Knowledge Base (KB)

The `KB/` directory contains all **knowledge base documentation** for the `vibe-dev-kit` repository.

## Structure

- `Architecture/` – Technical standards, ADRs, and integration docs
- `Changelog_and_Release_Notes/` – Changelogs and release notes
- `PM_and_Portfolio/` – Project management, Kanban, and governance

Additional documentation currently lives alongside `KB/` at the root:

- `kanban/` – Framework Kanban package docs
- `rituals/` – Additional ritual/policy docs

## Navigation

- **Architecture:** `KB/Architecture/`
- **Changelogs:** `KB/Changelog_and_Release_Notes/`
- **PM & Portfolio / Kanban:** `KB/PM_and_Portfolio/kanban/`

## Getting Started

- To understand dev-kit versioning: see `KB/Architecture/Standards_and_ADRs/dev-kit-versioning-policy.md`
- To understand Kanban governance: see `KB/PM_and_Portfolio/rituals/policy/kanban-governance-policy.md`
- To see project-level Kanban: see `KB/PM_and_Portfolio/kanban/_index.md`
